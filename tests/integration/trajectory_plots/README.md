## 기상학적 분석

### 2026-02-14 극동아시아 기상 패턴

**한반도 (서울, 부산, 제주):**
- 강한 남서풍 (20-26 km/h)
- 저기압 통과 패턴
- 북쪽으로 일관된 이동
- 겨울철 전형적인 기압 배치

**일본 (도쿄, 오사카):**
- 약한 동풍 (6-13 km/h)
- 고기압 가장자리
- 서쪽으로 이동 (한국 방향)
- 상대적으로 정체

**중국 (베이징, 상하이):**
- 대륙성 기단 영향
- 베이징: 서쪽으로 (대륙 내부)
- 상하이: 북쪽으로 (해안선 따라)

**대만 (타이베이):**
- 아열대 기단
- 동쪽으로 이동 (태평양 방향)
- 유일하게 동쪽 이동
